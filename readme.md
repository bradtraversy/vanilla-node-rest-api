# Vanilla Node REST API

> Full CRUD REST API using Node.js with no framework

```
# Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```

## Usage

```
# Install dependencies
npm install

# Run in develpment
npm run dev

# Run in production
npm start
```
